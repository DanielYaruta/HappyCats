# HappyCats 🐾

🔗 **[Демо на GitHub Pages](https://danielyaruta.github.io/HappyCats/)**

Лендинг приюта для животных: посетитель оставляет имя и почту, чтобы узнать первым, когда в приюте появятся котята, готовые к усыновлению.

Двухэкранная страница — баннер с призывом прокрутить вниз и форма подписки — со светлой/тёмной темой и валидацией полей на чистом JS.

## Технологии

- HTML / CSS (CSS-переменные, `aspect-ratio`, медиа-запросы)
- Vanilla JavaScript (без зависимостей, IIFE-модуль)
- Шрифты: Baloo 2, Nunito (Google Fonts)

## Функциональность

- Переключение светлой/тёмной темы с анимацией «отпечатка лапки»
- Плавный скролл от баннера к форме
- Валидация полей формы (имя — кириллица, почта — формат адреса) с инлайн-ошибками
- Заглушка успешной отправки (без реального бэкенда — задел под будущий fetch-запрос)
