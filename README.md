# 🐾 HappyCats

**Лендинг приюта для животных** — посетитель оставляет имя и почту,
чтобы узнать первым, когда в приюте появятся котята, готовые к усыновлению.

[**🔗 Демо на GitHub Pages**](https://danielyaruta.github.io/HappyCats/)

</div>

---

## О проекте

Двухэкранная страница с одной цельной фоновой иллюстрацией на весь
разворот: сверху — приветственный баннер с котятами на заборе, снизу —
форма подписки на фоне травы с играющими котятами.

- 🌗 Светлая/тёмная тема — переключаются одной кнопкой-лапкой, фон меняется вместе с темой (день ↔ ночь)
- ✍️ Валидация полей на чистом JS — без библиотек, с понятными подсказками об ошибках
- 🐱 Бренд-заголовок с лапкой внутри карточки формы
- 🔊 Звук мурлыканья как приятный бонус при успешной отправке формы
- 📱 Адаптивная вёрстка — фон не искажается ни на телефоне, ни на широком мониторе
- ⚡️ Без зависимостей — только HTML, CSS и vanilla JavaScript

## Технологии

- HTML / CSS — переменные, `aspect-ratio`, медиа-запросы, `background-clip: text`
- Vanilla JavaScript — без зависимостей, один IIFE-модуль
- Шрифты — Baloo 2, Nunito (Google Fonts)
